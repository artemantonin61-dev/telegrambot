# 🤖 Moderate Artem - Бот-модератор для Telegram

Полноценный бот-модератор с экономикой, статусами и премиум подпиской.

## 🌟 Возможности

### 🛡 Модерация
- Бан, разбан, мут, кик
- Предупреждения (автобан после 3)
- Очистка сообщений
- Антифлуд, антиспам, фильтр мата
- Фильтр ссылок и рекламы

### 💰 Экономика
- Внутренняя валюта **Ириски**
- Покупка статусов (Новичок, Гоблин, Король и др.)
- Премиум подписка за ириски
- Донат звёздами Telegram
- Ежедневный бонус

### 👑 Премиум подписка
- Админ-команды в любых чатах
- Удвоенный бонус
- Специальный статус
- Приоритетная поддержка
- +50% к рефералам

### 🔗 Дополнительно
- Реферальная система
- Рассылки для админа
- Поддержка пользователей
- Политика и оферта
- Статистика
