# Telegram Бот

Простой Telegram бот с разными функциями.

## Возможности
- 👤 Профиль пользователя
- 📝 Заметки
- 🎮 Игры (Камень-ножницы-бумага, Орёл-решка, Угадай число)
- 💰 Курсы валют
- 🕒 Время и дата
- 🎲 Случайные числа

## Команды
- `/start` - Запуск бота
- `/help` - Помощь

## Технологии
- Python 3
- pyTelegramBotAPI
