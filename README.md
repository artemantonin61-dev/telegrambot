# 🤖 Бот-модератор для Telegram групп

Мощный бот для модерации чатов, аналог Iris.

## Возможности

### 🛡 Модерация
- Бан / Разбан
- Мут / Анмут
- Кик
- Предупреждения
- Очистка сообщений

### 🔍 Фильтры
- Антифлуд
- Антиспам (ключевые слова)
- Фильтр ссылок
- Фильтр мата
- Автоудаление рекламы

### 👋 Управление
- Приветствие новых участников
- Правила чата
- Статистика
- Логирование действий

## Команды для админов

| Команда | Описание |
|---------|----------|
| /ban [причина] | Забанить пользователя |
| /mute [мин] [причина] | Ограничить пользователя |
| /unmute | Снять ограничение |
| /warn [причина] | Выдать предупреждение |
| /clear [N] | Удалить N сообщений |
| /settings | Настройки чата |
| /rules | Показать правила |
| /setrules [текст] | Установить правила |
| /setwelcome [текст] | Установить приветствие |
| /stats | Статистика чата |

## Установка

1. Добавьте бота в группу
2. Выдайте права администратора
3. Настройте фильтры через /settings

## Технологии
- Python 3
- pyTelegramBotAPI
- SQLite
